##  Tech Stack

### **Core Frontend**
- **React 19**: Modern UI component architecture.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS 4**: Utility-first styling for complex, responsive layouts.

### **Animation & Experience**
- **Framer Motion**: Advanced physics-based animations and scroll-linked transitions.
- **Lenis**: Smooth inertial scrolling for an elite "app-like" feel.
- **Spline**: Interactive 3D scenes integrated directly into the browser.

### **Icons & Assets**
- **Lucide React**: Clean and tree-shakable SVG iconography.
- **React Icons**: Standardized technology logos for the tech stack orbit.

##  Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/urvashivankar/Urvashi-portfolio.git
cd Urvashi-portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

##  Folder Structure

```text
├── public/              # Static assets & PDF resumes
├── src/
│   ├── components/      # UI Components (Hero, About, Projects, etc.)
│   ├── context/         # Theme and Global State
│   ├── assets/          # Project images and SVGs
│   ├── App.jsx          # Main application entry point
│   └── index.css        # Global styles & Tailwind directives
├── package.json         # Project dependencies
└── vite.config.js       # Vite configuration
```

---

##  Customization

To personalize this portfolio:
1. **Details**: Update info in `src/components/Experience.jsx`, `About.jsx`, and `Hero.jsx`.
2. **Projects**: Add your work to `src/components/Projects.jsx`.
3. **Styles**: Modify colors in `tailwind.config.js` or `index.css`.
4. **Resume**: Replace the PDF in `public/Fullstack_urvashi_RESUME.pdf`.

## 🚢 Deployment

The project is pre-configured for **Vercel** or **Netlify**.

1. Push your code to GitHub.
2. Connect your repository to Vercel.
3. The build settings are automatic:
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
